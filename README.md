# ResumeGen

A desktop application for tailoring resumes for every job application from your master experience database.

## Overview

The application lets you maintain a single source of truth for all your work history, education, projects, skills, etc and use it to customize resumes for each job application.
Tag your experiences and projects with keywords. New generated resumes will be use a job posting's keywords and match it to your own experiences.
Edit and fine-tune every detail with a real time PDF preview before exporting to a file.


Uses SQLite to keep all of your personal data in your own machine.
HTML/CSS templates used for generation can be customized to your own needs.
